## Getting started

### Installation:

1. Clone repo:

   ```shell
       git clone https://github.com/PrimeRogue/zola-mobile.git
   ```

2. Install dependency:

   ```shell
       npm install
   ```

3. Run server: http://localhost:19006/:
   ```shell
       npm start
   ```
